Week 01 Notes

    UX = User Experience UI = User Interface

    “The simple guideline is whatever you are doing – do mobile first,” -Eric Schmidt, World Mobile Congress Keynote 2012 Speed Matters

    Layout
  • Reduce image usage throughout sites/apps
    • SVG/Webfonts are lower bandwidth
  • Optimize and minify CSS and JS files
    • Remove unused tags and whitespace
  • GZIP files to eliminate unnecessary data


    • UX Menus
  • Keep the most used options/icons/apps in the easiest to reach places (the bottom in most cases, thinks Amazon app)


    • Forms
  • Make passwords visible by default on mobile. It reduces errors and when passwords are hidden, they do not add any security to the device.
  • CC numbers should be spaced so the user can easily read/write the number in the field.
  • When a number is needed to be entered use the number pad instead of the default keyboard.


    • Advertisements
  • No one wants to visit a website that has 40%+ ads (think any recipe or “clickbait” ad)
  • Stop using Pop-ups/Pop-overs
  • Content should come first
